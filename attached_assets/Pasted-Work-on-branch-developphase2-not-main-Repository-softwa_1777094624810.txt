Critical unit rule:
models.json stores API costs per 1K tokens, not per 1M tokens.

So:
- $5.00 per 1M input tokens = 0.005 per 1K input tokens
- $30.00 per 1M output tokens = 0.03 per 1K output tokens
- $0.15 per 1M input tokens = 0.00015 per 1K input tokens

Do not confuse per-1M and per-1K pricing.
