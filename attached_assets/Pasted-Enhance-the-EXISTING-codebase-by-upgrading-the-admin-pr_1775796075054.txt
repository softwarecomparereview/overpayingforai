## Current state

The current admin pricing refresh flow only supports:

* sample candidate generation
* pasted JSON candidate import

## Goal

Add live pricing retrieval for selected providers so the admin workflow can:

1. fetch latest provider pricing from official sources
2. normalize it into the existing pricing candidate format
3. compare current vs fetched candidate values
4. show before-vs-after diffs in the existing admin review UI
5. still require manual approval before updating/exporting
