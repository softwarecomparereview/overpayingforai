# Overpaying for AI

Static-first React + Vite app for comparing AI costs, subscriptions, and model choices.

## Build

```bash
pnpm --filter @workspace/overpaying-for-ai run build
```

## Deploy

Publish the static output from `artifacts/overpaying-for-ai/dist/public`.
Use that folder for GitHub Pages, Cloudflare Pages, or any static host.
